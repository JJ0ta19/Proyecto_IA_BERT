# 5. skills_taxonomy/

## Dataset
Skills Dataset / ESCO Skills Taxonomy

## Finalidad
Construir una base estructurada de habilidades profesionales para mejorar:
- extracción de skills
- matching laboral
- recomendaciones inteligentes

## Qué aprenderá la red neuronal

### Taxonomía de habilidades
- Clasificar skills en categorías jerárquicas (padre → hijo)
- Distinguir entre skills técnicas, blandas y de dominio
- Manejar miles de skills únicos con sus relaciones

### Extracción de skills mejorado
- Detectar menciones de skills aunque estén escritas de forma diferente
- Normalizar skills a estándar de la taxonomía (React.js → React)
- Identificar skills implícitas basadas en experiencia mencionada

### Relaciones entre habilidades
- Identificar skills que siempre van juntas (SQL + Bases de datos)
- Detectar skills que son prerequisito de otras
- Mapear clusters de skills para profesiones específicas

### Mapping tecnología - profesión
- asociar frameworks/herramientas a profesiones (TensorFlow → Data Scientist)
- Identificar skills transversales vs especializadas
- Entender qué tecnologías son substitutas entre sí

### Recomendación basada en skills
- Sugerir skills complementarias a las que el usuario tiene
- Recomendar profesiones basadas en skillset actual
- Identificar gaps de skills para transición de carrera

### Normalización de texto
- Estandarizar nombres de skills a formato consistente
- Manejar acrónimos y nombres completos (JS → JavaScript)
- Unificar variations de una misma skill

```text
Skill Extraction + Skill Matching
```
