# 4. resume_large_scale/

## Dataset
54k Resume Dataset

## Finalidad
Mejorar la generalización y robustez del modelo.

## Qué aprenderá la red neuronal

### Generalización a profesiones diversas
- Reconocer más de 100+ categorías profesionales diferentes
- Manejar profesiones niche o especializadas (ingeniero de звук, Especialista en compliance, etc.)
- Evitar overfitting a profesiones comunes

### Variedad de estilos de CV
- Procesar resumes en diferentes formatos y estructuras
- Adaptarse a diferentes estilos de escritura (formal, casual, técnico)
- Manejar diferentes longitudes de CV (1 página vs 5 páginas)

### Diversidad de industrias
- Comprender vocabulario específico de múltiples industrias
- Adaptar análisis según sector (tech, salud, legal, educación, etc.)
- Manejar diferencias regionales en terminología laboral

### Robustez del modelo
- Mantener accuracy alto con datos nunca vistos
- Manejar CVs con ruido o imperfecciones
- Evitar sesgos hacia profesiones sobre-representadas

### Balanceo de datos
- Aprender de distributions desbalanceadas de profesiones
- Aplicar técnicas de data augmentation para clases minoritarias
- Evitar que profesiones dominantes dominen el modelo

```text
Entrenamiento masivo y robustez del modelo
```
